## SUI-CS – Flash Loan Arbitrage on Sui

SUI-CS is a decentralized application for executing **flash loan and arbitrage** operations on the **Sui** network.
It leverages the **Cetus CLMM**, **Turbos CLMM** liquidity pools and the **DeepBook (DEEP token)** order book to seek arbitrage opportunities between the **SUI / DEEP** pairs.

The platform borrows an amount of SUI tokens from a liquidity pool on the Cetus DEX. These SUI are swapped for DEEP on Cetus, then swapped back for SUI on the Turbos DEX. The user pockets the difference.

The transaction can be cancelled if the profits generated by the arbitrage do not cover the transaction fees for the various operations.

This project was built as part of the **SUI x ESILV** hackathon organized by ESILV in Paris La Défense in November 2025.

## Getting Started
Navigate to the `frontend-code` folder.

Run the following command in your terminal: `npm run dev`

Open the app locally at: `http://localhost:5173/`.

---
## Key Features

- **SUI flash loan** on a Cetus pool (SUI / DEEP).
- **SUI → DEEP swap** on Cetus via flash swap.
- **DEEP → SUI swap** on Turbos CLMM.
- **Automatic repayment** of the flash loan (principal + fees) within the same transaction.
- **Profit transfer** to the user's wallet if the arbitrage is profitable.
- React web interface:
  - Wallet connection via `@mysten/dapp-kit`.
  - Pool selection (SUI / DEEP).
  - Flash loan amount input.
  - Transaction tracking (digest + SuiScan link).
  - User-friendly error messages (unprofitable arbitrage, deadline, network issues, etc.).

## Prerequisites

- **Node.js** (recent version, ≥ 18 recommended).
- **pnpm / npm / yarn** (your choice, examples use `npm` below).
- **Sui CLI** installed (to build & publish the Move package).
- A **dapp-kit compatible Sui wallet** (e.g. Sui Wallet, Ethos, Phantom, etc.) with SUI on the target network (testnet or mainnet).
